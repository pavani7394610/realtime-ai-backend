# Realtime AI Backend (FastAPI + WebSockets)

## Overview
This project is a realtime backend built using FastAPI and WebSockets.
It supports session-based WebSocket connections, message handling,
and lightweight persistence using SQLite.

This backend is suitable for realtime chat, AI session handling,
or event-based communication systems.

## Tech Stack
- Python 3.13
- FastAPI
- WebSockets
- Uvicorn
- SQLite
